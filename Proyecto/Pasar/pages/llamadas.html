Llamadas

General.

Descartes Basicos

Cuando el usuario llama, ya sea porque no recibe o no puede realizar llamadas, realizar el siguiente paso a paso:

  • Verificar que la linea este activa
  • Validar si el problema es de llamadas de entrada o salida
  • Brindar codigos de restriccion o desvio
  • Probar en otro equipo
  • Escalar el caso
×

Inconvenientes.

Estos son los casos que ingresan con inconvenientes que se presenta generalmente.

Aqui encontraras el tipo de problemas y los descartes que se deben realizar

Problema Descarte
Buzon de Mensajes Verificar que el IMEI no este bloqueado. Probar en otro equipo.
Usuario no se encuentra disponible Verificar que el IMEI no este bloqueado, codigos, linea PreActive
Mal Enrutada Linea portada, suena tururu o red no disponible
Real Balance Con saldo, dice no tiene saldo o esta en mora.
No salen llamadas Validar que tenga el saldo, verificar marcado, probar en otro equipo.

Escalamientos.

Escalamiento a los proveedores

Los escalamientos con problemas de llamadas se deben realizar a TIGO o a REDKNEE segun el tipo de caso En la siguiente tabla se podra encontrar de que forma se puede realizar segun corresponda.

Para escalar a TIGO, se escala al correo electrónico Soporte.Mvno@TigoUne.com, con el consecutivo.
Para escalar a REDKNEE se realiza a traves de SPIDER.

Estos casos se escalan segun corresponda:

Aqui encontraras el tipo de escalamientos

Problema Proveedor Como se escala
Buzon de Mensajes TIGO Descripcion de los descartes realizados.
Usuario no se encuentra disponible REDKNEE Realziarle ActiveMobile
Mal Enrutada TIGO Verificar el enrutamiento de la linea
Real Balance REDKNEE Realziarle Real Balance
No salen llamadas TIGO Descripcion de los descartes realizados.

Nota: Dejar numero alterno!!
