General

El soporte técnico o asistencia técnica es un rango de servicios por medio del cual se proporciona asistencia a los usuarios al tener algún problema al utilizar un producto o servicio.

Navegacion

Cuando el usuario se comunica con problemas de navegacion, ya sea que no navega o que navega lento, seguir el siguiente paso a paso:

Paso a paso Verificar que tenga Datos Verificar que tenga los datos encendidos Configurar APN: movilexito.net.co Configurar Modo de Red: GSM/WCDMA Validar que el IMEI no se encuentre bloqueado Probar en otro equipo Escalar el caso

Informacion

Este tipo de informacion puede ayudar cuando se esta realizando un descarte de problemas con la navegacion.

En la parte superior, cuando estan los datos encendidos, aparecen estas redes:

Letra Red Velocidades
G GPRS 2.5G 56~114kbps
E EDGE 2.8G 400kbps
H HSPA 3G+ 14.4Mbps
H+ HSPA+ 3.8G 168Mbps
LTE 4G 326,5Mbps

Por que navegamos lento?

Navegamos lento por muchas razones asociadas a distintos factores, que causan una navegacion lenta o que persivamos que no es tan rapida como queremos. Estos son algunos motivos:

  • Configuracion del equipo
  • Telefono no soporte la tecnologia
  • Muchas aplicaciones abiertas
  • Depende de la Infraestructura (Cobertura)
  • En Search busca la referencia del equipo
  • En NETWORK presiona Expand
  • En 4G bands busca 4(1700/2100)
  • Si no aparece es porque el equipo no soporta esta tecnologia.
  • No todas las referencias de equipos soportan 4G.
Nota! Dejar numero alterno.

Llamadas

Cuando el usuario llama, ya sea porque no recibe o no puede realizar llamadas, realizar el siguiente paso a paso:

Paso a paso Verificar que la linea este activa Validar si el problema es de llamadas de entrada o salida Brindar codigos de restriccion o desvio Probar en otro equipo Escalar el caso
Problema Descarte
Buzon de Mensajes Verificar que el IMEI no este bloqueado. Probar en otro equipo.
Usuario no se encuentra disponible Verificar que el IMEI no este bloqueado, codigos, linea PreActive
Mal Enrutada Linea portada, suena tururu o red no disponible
Real Balance Con saldo, dice no tiene saldo o esta en mora.
No salen llamadas Validar que tenga el saldo, verificar marcado.
Nota! Dejar numero alterno.
